* 📋 Overview

Modern, AI-powered development environment for macOS and Linux. Managed by [[https://www.chezmoi.io/][chezmoi]] with [[https://github.com/jdx/mise][mise]] for runtime/package management, [[https://nixos.org/][Nix]] for system-level reproducibility, and extensive editor integrations.

#+begin_quote
*Philosophy:* Modular, reproducible, AI-assisted development with literate configuration.
#+end_quote

** 🎯 Key Features

- ⌨️ Multi-layout keyboard support — [[#keyboard-layouts][Colemak (HNEI) and QWERTY (HJKL)]]
- 💻 *Zellij* / *cmux* for rich terminal multiplexing
- 🐚 *Xonsh* - Python-powered shell with rich customization
- 📦 *mise* - Declarative runtime & global package management
- 🏗️ *Nix + Home Manager* - Reproducible system configuration
- ✨ *Multi-editor* - Emacs (primary), Helix for quick terminal edits
- 🤖 *AI Integration* - Pi, ECA, OpenCode for intelligent coding assistance
- 🎨 *Desktop* - macOS: Yabai / Linux: Niri + Hyprland
- 🔧 *Polyglot* - Go, Rust, Node.js, Python, Java tooling

* 🚀 Quick Start

** Prerequisites

- macOS (Apple Silicon or Intel) or Linux (Fedora)
- [[https://www.chezmoi.io/][chezmoi]] installed

** Bootstrap Installation

#+NAME: install-chezmoi
#+BEGIN_SRC bash
# Install chezmoi (macOS)
brew install chezmoi
# Or Linux
curl -sfL https://get.chezmoi.io | sh

# Apply dotfiles
chezmoi init --apply https://github.com/Artawower/dotfiles
#+END_SRC

** Install Tools

#+NAME: install-tools
#+BEGIN_SRC bash
# Full setup (includes mise, nix, cargo tools, etc.)
just init-mac     # macOS
just init-linux   # Linux

# Or step by step:
just mise         # Install runtimes + npm globals via mise
just uv           # Python tools via uv
just cargo        # Rust tools
just go           # Go tools (gopls)
#+END_SRC

** Install Nix (optional but recommended)

#+NAME: install-nix
#+BEGIN_SRC bash
# Install Nix package manager (multi-user installation)
curl --proto '=https' --tlsv1.2 -sSf -L https://install.determinate.systems/nix | \
  sh -s -- install

# Source nix
. /nix/var/nix/profiles/default/etc/profile.d/nix-daemon.sh

# Apply Nix configuration
just nix-mac      # macOS
just nix-linux    # Linux
#+END_SRC

** Post-Installation

#+NAME: post-install
#+BEGIN_SRC bash
# Configure Xonsh as default shell
echo "$(which xonsh)" | sudo tee -a /etc/shells
chsh -s $(which xonsh)

# Switch keyboard layout (default: colemak)
just layout qwerty  # or "just layout" to pick interactively
#+END_SRC

* ⌨️ Keyboard Layouts
:PROPERTIES:
:CUSTOM_ID: keyboard-layouts
:END:

All configs support both **Colemak** (HNEI) and **QWERTY** (HJKL) layouts via a single variable in =.chezmoidata.toml=:

#+BEGIN_SRC toml
layout = "colemak"   # or "qwerty"
#+END_SRC

Switch with one command:

#+BEGIN_SRC bash
just layout qwerty   # switch to QWERTY
just layout colemak  # switch back
just layout          # pick interactively with fzf
#+END_SRC

* 🎨 Customization
** Xonsh Aliases
60+ aliases configured in =xonsh/aliases.xsh=:
- Docker shortcuts (=dps=, =drm=, =dex=)
- Kubernetes (=k=, =kgp=, =kd=)
- Git operations (=gst=, =gco=, =gp=)
- Development (=pn= for pnpm, =bn= for bun)

* 🔧 Management with Chezmoi

This repo is a [[https://www.chezmoi.io/][chezmoi]] source state (=chezmoi edit= opens files from =~/.local/share/chezmoi/=).

| Command              | Purpose                          |
|----------------------+----------------------------------|
| =chezmoi apply=       | Deploy changes to =~/=            |
| =chezmoi edit <file>= | Edit source version of a file    |
| =chezmoi diff=        | Preview pending changes          |
| =chezmoi re-add=      | Sync target → source             |
| =just mise=           | Install/update mise packages     |
| =just layout=         | Switch keyboard layout           |

OS-specific configs are excluded via =.chezmoiignore.tmpl=:
- Linux configs (niri, hyprland, xremap, cava, noctalia...) skipped on macOS
- macOS configs (yabai, karabiner, sketchybar, skhd...) skipped on Linux
