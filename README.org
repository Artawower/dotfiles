* 📋 Overview

Modern, AI-assisted development environment for macOS and Fedora Asahi Linux. The repository is managed by [[https://www.chezmoi.io/][chezmoi]], uses [[https://github.com/jdx/mise][mise]] for runtimes and global developer tools, and keeps reproducible system/user configuration in [[https://nixos.org/][Nix]], Home Manager, and nix-darwin modules.

#+begin_quote
*Philosophy:* modular, reproducible, keyboard-driven, AI-assisted development across macOS and Linux.
#+end_quote

** 🎯 Key Features

- ⌨️ Multi-layout keyboard support — [[#keyboard-layouts][Colemak HNEI and QWERTY HJKL]]
- 🐚 *Xonsh* — Python-powered shell split into small modules
- 💻 *Zellij* / *cmux* — terminal multiplexing and AI-native terminal workflows
- 📦 *mise* — declarative runtimes and global tools
- 🏗️ *Nix + Home Manager + nix-darwin* — reproducible packages and host integration
- ✨ *Editors* — Emacs as the main IDE, Helix for fast terminal editing, Zed config included
- 🤖 *AI tooling* — Pi, ECA, OpenCode, local skills/agents
- 🎨 *Desktop* — macOS: Aerospace/skhd/Karabiner/SketchyBar; Linux: Niri with Hypr/Noctalia/Vicinae support
- 🔧 *Polyglot* — Node.js/frontend, Python, Go, Rust tooling

* 🚀 Quick Start

** Prerequisites

- macOS or Fedora Linux
- Internet connection
- A working shell with =sh= and =curl=

** Bootstrap dotfiles

#+NAME: install
#+BEGIN_SRC bash
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply https://github.com/Artawower/dotfiles
#+END_SRC

This installs chezmoi, clones this repository as the source state, and applies the dotfiles to =$HOME=.

** Install Nix

Nix is the recommended way to get the base toolchain, including =just=.

#+NAME: install-nix
#+BEGIN_SRC bash
curl --proto '=https' --tlsv1.2 -sSf -L https://install.determinate.systems/nix | \
  sh -s -- install

. /nix/var/nix/profiles/default/etc/profile.d/nix-daemon.sh
#+END_SRC

** Apply system/user configuration

After =just= is available, use the unified recipes from =~/.config/Justfile=:

#+NAME: install-tools
#+BEGIN_SRC bash
cd ~/.config
just init     # first-time OS bootstrap
just nix      # apply nix-darwin/Home Manager config
just mise     # install mise runtimes and global tools
#+END_SRC

For regular maintenance:

#+BEGIN_SRC bash
cd ~/.config
just update   # update OS packages, Nix/Home Manager, mise, uv, Pi extensions
just sync     # update + re-add flake lock + bump mise-managed versions
just clean    # garbage collection and cleanup
just doctor   # environment diagnostics
#+END_SRC

** Post-installation

#+NAME: post-install
#+BEGIN_SRC bash
# Configure Xonsh as default shell, if desired
echo "$(which xonsh)" | sudo tee -a /etc/shells
chsh -s "$(which xonsh)"

# Switch keyboard layout
cd ~/.config
just layout qwerty
just layout colemak
just layout          # pick interactively with fzf
#+END_SRC

* ⌨️ Keyboard Layouts
:PROPERTIES:
:CUSTOM_ID: keyboard-layouts
:END:

Most interactive configs support both **Colemak** (HNEI) and **QWERTY** (HJKL). The active layout is stored in =.chezmoidata.toml=:

#+BEGIN_SRC toml
layout = "colemak"   # or "qwerty"
#+END_SRC

Switch with:

#+BEGIN_SRC bash
cd ~/.config
just layout qwerty
just layout colemak
just layout
#+END_SRC

Layout-sensitive files include Helix keymaps, xonsh keybindings, xremap, Karabiner, Niri, skhd, and Zellij configs.

* 🔧 Management with Chezmoi

This repository is a [[https://www.chezmoi.io/][chezmoi]] source state. Source files live in =~/.local/share/chezmoi/= and deploy into =$HOME=.

| Command                         | Purpose                         |
|---------------------------------+---------------------------------|
| =chezmoi apply=                 | Deploy source state to =$HOME=  |
| =chezmoi edit <target-file>=    | Edit source version of a target |
| =chezmoi diff=                  | Preview pending target changes  |
| =chezmoi re-add <target-file>=  | Sync target changes back to source |
| =cd ~/.config && just init=     | Bootstrap current OS            |
| =cd ~/.config && just update=   | Update OS/Nix/toolchains        |
| =cd ~/.config && just layout=   | Switch keyboard layout          |

OS-specific configs are excluded via =.chezmoiignore.tmpl=:

- Linux-only configs are skipped on macOS: niri, xremap, hypr, cava, noctalia, vicinae, systemd.
- macOS-only configs are skipped on Linux: aerospace, skhd, karabiner, sketchybar, yabai, launchagents.

* 🔧 System Requirements

- *OS:* macOS / Fedora Linux
- *Arch:* Apple Silicon and ARM64 Linux are first-class in the current Nix hosts; Intel may require small adjustments
- *Disk:* several GB for Nix store, language runtimes, and editor tooling
- *Network:* required for bootstrap and package updates
