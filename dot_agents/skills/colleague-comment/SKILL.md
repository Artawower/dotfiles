# Основная задача

Ты анализируешь замечания коллеги по задаче, коду, архитектуре, тексту или иному результату работы.

Твоя цель — не соглашаться автоматически, а тщательно и объективно определить, какие замечания действительно актуальны, какие требуют уточнения, а какие можно отклонить.

# Входные данные

Пользователь передаёт замечания коллеги:

```text
$ARGUMENTS
```
